## API Reference

### compute_cid(content)

Computes a content identifier from binary data.

**Parameters:**
- `content` (bytes): The content to hash

**Returns:** `str` - The computed CID

**Example:**
```python
cid = compute_cid(b"test content")
print(cid)  # AAAAAAAAAAAAAAAAEG5lc3QgY29udGVudA
```

**Implementation:**
```python
def compute_cid(content: bytes) -> str:
    prefix = encode_length(len(content))
    if len(content) <= 64:
        suffix = to_base64url(content)
    else:
        suffix = to_base64url(hashlib.sha512(content).digest())
    return prefix + suffix
```

<Note>
  Content ≤64 bytes is embedded directly. Larger content uses SHA-512 hashing.
</Note>

### download_cid(base_url, cid)

Downloads content from a URL and validates its CID.

**Parameters:**
- `base_url` (str): Base URL (e.g., `'https://256t.org'`)
- `cid` (str): The content identifier to fetch

**Returns:** `Tuple[bytes, str, bool]`
- `content`: Downloaded content as bytes
- `computed`: CID computed from downloaded content
- `is_valid`: True if computed matches expected CID

**Example:**
```python
try:
    content, computed, is_valid = download_cid("https://256t.org", cid)
    if is_valid:
        print(f"Downloaded: {content.decode()}")
    else:
        print(f"CID mismatch! Got: {computed}")
except Exception as e:
    print(f"Download failed: {e}")
```

**Implementation:**
```python
def download_cid(base_url: str, cid: str) -> Tuple[bytes, str, bool]:
    """
    Download content from a URL and validate its CID.
    
    Args:
        base_url: Base URL to download from (e.g., "https://256t.org")
        cid: Expected content identifier
        
    Returns:
        Tuple of (content, computed_cid, is_valid)
    """
    url = f"{base_url.rstrip('/')}/{cid}"
    req = Request(url, headers={'User-Agent': 'Mozilla/5.0'})
    with urlopen(req) as response:
        content = response.read()
    computed = compute_cid(content)
    return content, computed, computed == cid
```

### Helper Functions

#### to_base64url(data)

Converts bytes to base64url encoding (URL-safe base64 without padding).

```python
def to_base64url(data: bytes) -> str:
    return base64.urlsafe_b64encode(data).decode().rstrip("=")
```

#### encode_length(length)

Encodes content length as a 6-byte base64url string.

```python
def encode_length(length: int) -> str:
    return to_base64url(length.to_bytes(6, "big"))
```

## Complete Example

Here's the full implementation with type annotations:

```python cid.py
import base64
import hashlib
from pathlib import Path
from typing import Tuple
from urllib.request import Request, urlopen


BASE_DIR = Path(__file__).resolve().parents[2]
EXAMPLES_DIR = BASE_DIR / "examples"
CIDS_DIR = BASE_DIR / "cids"


def to_base64url(data: bytes) -> str:
    return base64.urlsafe_b64encode(data).decode().rstrip("=")


def encode_length(length: int) -> str:
    return to_base64url(length.to_bytes(6, "big"))


def compute_cid(content: bytes) -> str:
    prefix = encode_length(len(content))
    if len(content) <= 64:
        suffix = to_base64url(content)
    else:
        suffix = to_base64url(hashlib.sha512(content).digest())
    return prefix + suffix


def download_cid(base_url: str, cid: str) -> Tuple[bytes, str, bool]:
    """
    Download content from a URL and validate its CID.
    
    Args:
        base_url: Base URL to download from (e.g., "https://256t.org")
        cid: Expected content identifier
        
    Returns:
        Tuple of (content, computed_cid, is_valid)
        - content: The downloaded content as bytes
        - computed_cid: The CID computed from the downloaded content
        - is_valid: True if computed_cid matches the expected cid
    """
    url = f"{base_url.rstrip('/')}/{cid}"
    req = Request(url, headers={'User-Agent': 'Mozilla/5.0'})
    with urlopen(req) as response:
        content = response.read()
    computed = compute_cid(content)
    return content, computed, computed == cid


__all__ = [
    "BASE_DIR",
    "EXAMPLES_DIR",
    "CIDS_DIR",
    "compute_cid",
    "download_cid",
]
```

## Performance Characteristics

<Info>
  The Python implementation offers:
  - **Small files** (less than 64 bytes): ~10 microseconds
  - **Large files**: Limited by SHA-512 performance (~500 MB/s)
  - **Memory efficient**: Constant memory for hash computation
</Info>
