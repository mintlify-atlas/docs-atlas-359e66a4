The 256t.org specification has been implemented in over 40 programming languages. Each implementation provides the same core functionality: computing and verifying content identifiers.

    ```rust Rust
    let content = b"Hello, World!";
    let cid = compute_cid_from_bytes(content);
    ```

    ```ruby Ruby
    content = "Hello, World!"
    cid = compute_cid(content)
    puts cid
    ```

    ```java Java
    byte[] content = "Hello, World!".getBytes();
    String cid = CidUtil.computeCid(content);
    System.out.println(cid);
    ```
    
    ```bash Bash
    source cid.sh

    cid=$(compute_cid /tmp/test)
    echo "$cid"
    ```

## Contributing

Want to add a new language? Each implementation needs:

1. **Core functions**: `compute_cid()` and `download_cid()`
2. **Helper functions**: `encode_length()` and `to_base64url()`
3. **Test scripts**: `generate` and `check` scripts
4. **Documentation**: README with usage examples

See the [contribution guide](https://github.com/curtcox/256t.org/blob/main/CONTRIBUTING.md) for details.
